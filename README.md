# 棠想視界 KIKI DESIGN (2026)

> **Create with Soul.** — 視覺設計・品牌規劃・數位創作

🔗 **線上作品集**：[https://kiki-design.vercel.app/](https://kiki-design.vercel.app/)

Kiki Design 以極簡高對比視覺語言為核心，結合 React + GSAP 打造高效能、可維護、可持續演進的數位作品展示平台。

---

## 專案亮點

- 極簡主義視覺系統（黑白灰 + 細邊框節奏）
- GSAP / ScrollTrigger 動態敘事與流暢捲動體驗
- Mobile First 響應式佈局與 CLS 防禦策略
- TypeScript 嚴格型別與資料驅動渲染（SSOT）

---

## 技術棧

- React 18
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Vite

---

## 本機開發

```bash
npm install
npm run dev
npm run build
npm run preview
```

---

## 專案結構

```text
kikidesign/
├── src/                 # 應用程式主程式與樣式
├── public/              # 靜態資源
├── .cursor/rules/       # AI 規範卡（00~05）
├── index.html           # HTML 入口
├── package.json         # scripts / dependencies
└── vite.config.ts       # Vite 設定
```
