## 關於此專案

這是我的個人視覺設計作品集網站，用於展示品牌識別、平面設計與網頁視覺等作品。

網站由我獨立設計與開發，從視覺風格到程式架構皆親自完成。

---

## 技術架構

- **框架：** React 18 + TypeScript
- **建置工具：** Vite
- **動畫：** GSAP + ScrollTrigger
- **樣式：** Tailwind CSS + 自訂 CSS 變數
- **部署：** GitHub Pages

---

## 本機執行

```bash
# 安裝套件
npm install

# 開發模式
npm run dev

# 建置
npm run build

# 部署至 GitHub Pages
npm run deploy
```
